# Atom常用快捷键

Atom 专门有个模块列出所有的快捷键

```
# 快捷键
Ctrl + /                启用注释
Ctrl + \                展示隐藏目录树
Ctrl + Alt + I          打开Chrome调试器
Ctrl + [                向右缩进
Ctrl + ]                向左缩进
Shift + Home            选定光标至行首
Shift + End             选定光标至行尾
Ctrl + D                匹配选定下一个
Alt + F3                匹配选定所有
Ctrl + up               选中行上移
Ctrl + down             选中行下移
​
# 目录树操作
a   添加文件
d   将当前文件另存为
i   显示(隐藏)版本控制忽略的文件
​
# 折叠
Alt + Ctrl + [          折叠
Alt + Ctrl + ]          展开
Alt + Ctrl + Shift + {  折叠全部
Alt + Ctrl + Shift + }  展开全部
​
# Markdown 写作
Ctrl + Shift + M        Markdown预览
​
# Markdown 语法补全
b, legal, img, l, i, code, t, table
```
